About Me

Hi, I’m Cortland Burns! Over the last two years I’ve grown from building single-app solutions to delivering secure, production-ready systems across web and mobile. My focus is application development, full-stack design, and secure architecture—with hands-on work hardening auth flows, validating inputs, and automating builds/tests. For mobile Flutter projects I integrate cloud services; the Travlr (MEAN) project remains vendor-agnostic.

Automation & Security Setup (Quick Commands)

MEAN API Hardening

npm i helmet express-rate-limit cors bcrypt jsonwebtoken dotenv zod validator express-mongo-sanitize compression
// Express snippet
app.use(helmet());
app.use(cors({ origin: ["https://your-site.app"], methods: ["GET","POST","PUT","DELETE"], credentials: true }));
app.use(rateLimit({ windowMs: 15*60*1000, max: 200 }));

Angular Admin (JWT Interceptor)

ng g interceptor auth/jwt
// attach token, redirect on 401/403

CI with GitHub Actions (skeleton)

# .github/workflows/ci.yml
name: CI
on: [push, pull_request]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: 20 }
      - run: npm ci && npm run lint --if-present && npm test --if-present
